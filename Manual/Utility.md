## Embed Fixer
This sub-module is in charge of automatically replacing the Embeds for certains sites.  
__Compatible Websites__:
- Twitter
- Pixiv
-# This command can be called by the user application message menu.

### PixivID from Attachments
This is a Message Menu command that will look for Pixiv IDs in the file names of the selected message, if any is found, the Pixiv Artworks Handler is triggered and sends the fixed Pixiv Embed for the Artwork IDs found in the attachments' names.
-# This command is available without needing to install Kosaka as a Discord Application.

## First Message
`/first_message` - [All] Sends a message link to the first message in the channel.
